SVG

SVG står for Scalable Vector Graphics. Det er skalerbart, du kan få det til å bli stort og liten uten å bli blurry. 2D vektorbasert bildeformat. Lesbar XML (extended markup language) kode. Man kan manipulere SVG manuelt, med CSS eller Javascript. Vi kan bruke SVG når vi skal lage Logoer, ikoner, animasjoner, illustrasjoner, skjemaer/grafer. Man kan lage SVG ved å kode for hånd eller bruke bildebehandlingsprogammer som støtter vektorgrafikk. Eksempler på programmer man kan bruke er Inkscape, Adobe illustrator og Sketch.
